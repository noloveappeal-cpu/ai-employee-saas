# AI Employee SaaS

A production-ready starter for building your own AI Employee platform.

## Features
- Next.js 15 (App Router)
- AI generation API (OpenAI / Anthropic)
- Persona engine for high-conversion content
- Ready for Stripe subscriptions
- Ready for Supabase auth + database
- Clean file structure for scaling into a full SaaS

## Quick Start

```bash
git clone https://github.com/noloveappeal-cpu/ai-employee-saas
cd ai-employee-saas
npm install
cp .env.example .env
npm run dev
